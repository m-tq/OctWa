What is OctWa?

A secure browser-based wallet for the Octra blockchain network. Available as both a web application and Chrome/Edge browser extension.

OctWa supports two operation modes: Public Mode for standard on-chain transactions, and Private Mode for fully encrypted transactions powered by Octra HFHE. You control when your data is public or encrypted.

Key Features

Everything you need for public and private transactions.

How It Works

Two modes. One wallet. You control the visibility.

Public Mode

Standard blockchain transactions. Balances and transfer amounts are visible on-chain. Use for everyday transactions where transparency is acceptable or required.

Private Mode

Encrypted execution using Fully Homomorphic Encryption (FHE). Balances and amounts remain confidential. Transactions are processed without revealing sensitive data.

Security & Philosophy

Your keys. Your control. Always encrypted.

Keys Stay Local Private keys never leave your device. All encryption happens locally.
Encrypted-First Design AES-256-GCM encryption. PBKDF2 with 310,000 iterations. Zero unencrypted storage.
User-Controlled State You decide what's public or private. Encrypt or decrypt on demand.
Session Security Auto-lock after 15 min. Session encryption. Memory cleared on lock.
